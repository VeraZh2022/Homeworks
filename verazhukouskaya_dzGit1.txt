GIT Homework 1

Для выполнения задания у вас должен быть установлен для Windows - GitBash.
Создан аккаунт в GitHub

Все шаги сценария выполняйте в терминале GitBush, Terminal, в папке под гитом.

Как отправить ДЗ на проверку.
 1. Создайте текстовый файл как в первом ДЗ по Terminal.
 MacBook-Pro-13-Vera:3 github_terminal verazhukouskaya$ touch verazhukouskaya_dzGit

 2. Сценарий перенесите в этот файл.
 MacBook-Pro-13-Vera:3 github_terminal verazhukouskaya$ cat > verazhukouskaya_dzGit
 (скопировала сценарий в терминал, enter, ctrl+c)

 3. Напротив каждого действия - напишите команду в GitBash
 (вношу через atom, открывая сам файл txt)

Файл со сценарием и ссылку (мой гит https://github.com/VeraZh2022?tab=repositories) на свой гит хаб отправляйте менторам на проверку.
